Manage telemetry preferences

Usage: pybun telemetry [OPTIONS] <COMMAND>

Commands:
  status   Show current telemetry status
  enable   Enable telemetry (persisted)
  disable  Disable telemetry (persisted)
  help     Print this message or the help of the given subcommand(s)

Options:
      --format <FORMAT>  Output format for machine readability [default: text] [possible values: text, json]
      --telemetry        Enable telemetry for this invocation
      --no-telemetry     Disable telemetry for this invocation
  -h, --help             Print help
